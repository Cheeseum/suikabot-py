# suikabot
drunk oni irc robot

* python 2
* plugin based
* YAML configs
* pickled data storage

## Dependencies
* appdirs
* Twisted with SSL support
  * pyopenssl
  * pypiwin32 (on Windows)
* PyYAML
* humanize (for the `later` and `remind` plugins)
* pytz (for the `ping` plugin)
* parsedatetime (for the `remind` plugin)
